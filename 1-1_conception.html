<html> </html> ->이 문서는 html파일입니다. 확장자 : html

<head> </head> -> 창쪽에 뜨는 것 (탭근처)

<body> </body> -> 메인 화면에 뜨는 것


  • Html 태그, 헤드, 바디 테그 가 html의 뼈대를 이루고 온전한 html문서!
  • 파일 시작 부분에 <!DOCTYPE html>으로 문서 형식을 정의해줘야함. DTD 선언 -> 이 문서가 html인것을 명시
  • <meta charset="utf-8”> 은 이 문서의 인코딩 방식을 utf-8을 이용한다는 뜻
  • <h1>~<h6>까지 글자크기의 차이 -> 제목이기때문에 자동적으로 endl 됨.
  • <p>테그 : 한 문단을 쭉 치고 마지막에 endl됨 -> 문단 구분
  • <br>: bridge ==endl 줄바꿈 테그
  • <p>가 문단이라면 <span>은 문장!
  • <style> : 각각의 테그가 가질수 있는 속성 설정 <style="color:red">은 글자 빨간색

스타일 방식 : 태그를 이용해서 따로 쓰는 방법 OR 특정 태그 안에 넣어쓰는 방법

Ex) <style>

    h1 {color:red;}

    p {color:blue;}

  </style> 

//각각의 테그는 이름으로 구별할 수 있다!

  • <caption> : 표의 이름
  • <border> : 윤곽선 테두리의 굵기! 숫자가 커질수록 굵기가 굵어짐 0이라고 하거나 테그를 없애면 윤곽선이 사라짐
  • <thead>,<tbody>,<tfoot>이 있음
  • <tr>은 테이블 로우로 로우 먼저쌓고, column을 넣음.
  • <th> 는 굵고 가운데 정렬! <td>는 그냥 왼쪽 정렬
  • Colspan, rowspan -> 셀 병합기능. row는 줄, col은 세로줄 병합. 지정된 숫자 만큼을 합쳐줌.
